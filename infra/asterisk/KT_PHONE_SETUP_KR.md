# 1. 전체 흐름 설명

## 1.1 우리가 하려는 것

- 기존 KT 070 번호에 **AI 자동응답 시스템** 연결
- AI가 비영업시간 (저녁/주말/점심) 에 고객 전화를 자동 응대
- 고객에게는 **070 번호 대신 010 번호**로 표시되도록 설정 (스팸 의심 방지)
- KakaoTalk 메시지 채널과 병행 운영

## 1.2 두 가지 절차 동시 진행

| 구분 | 어디서 | 무엇을 | 소요일 | 비용 |
|---|---|---|---|---|
| **Part 1** | KT (1577-0114) | 070 SIP trunk 신청 | 1-3 영업일 | 월 사용료만 |
| **Part 2** | KCC (msafer.or.kr) | 발신번호 사전등록 (010 표시) | 2-3 영업일 | 무료 |

**두 가지 모두 완료해야 070이 완전히 숨겨집니다.** 같은 날 동시에 신청 가능.

## 1.3 왜 이렇게 해야 하나요?

- ❌ 070만 사용 → 고객이 스팸으로 의심, 통화율 낮음 (15-25%)
- ✅ 070 백엔드 + 010 표시 → 고객은 010만 보임, 통화율 높음 (60-80%)
- ⚖️ **법적으로 100% 합법** — 본인 명의의 두 번호이며, KCC 등록 후 정식 사용

## 2.3 010 번호가 개인 명의인 경우

KCC 발신번호 사전등록은 **표시 번호도 사업자등록증과 연결**되어야 합니다. 010이 개인 명의면:

### 방법 1: 법인/사업자로 명의변경 (권장)
- 통신사 고객센터에 전화 → "개인 → 법인 명의변경 신청"
- 가져갈 서류:
  - 사업자등록증
  - 양도양수 동의서 (개인 → 법인)
  - 대표자 신분증
  - (법인의 경우) 법인 인감증명서
- 소요일: 1-2 영업일

### 방법 2: 개인사업자 명의로 통합
- 법인이 아닌 **개인사업자**인 경우만 가능
- 같은 명의로 010 + 070 모두 등록되어 있으면 자동 인정

# 4. PART 2 — KCC 발신번호 사전등록 (070 숨기기)

## 4.1 무엇을 하는 절차인가요?

- **방송통신위원회 (KCC)** 가 운영하는 의무 등록 제도
- 출력 발신번호로 사용할 모든 번호를 사전 등록
- 등록 후, KT 가 우리 070 트렁크에서 발신 시 등록된 010 번호로 표시
- 한국 모든 콜센터·은행·배달앱이 사용하는 표준 절차

## 4.2 법적 근거

| 법령 | 내용 |
|---|---|
| **정보통신망법 §50-8** | 사기 목적의 발신번호 변조 금지. 합법적 사업자 표시는 허용. |
| **발신번호 사전등록제** (2015년 시행) | 발신번호 사전 등록 시 법적 표시 가능. |
| **전기통신사업법** | 통신사 (KT) 는 등록 발신번호와 실제 발신 일치 확인 의무. |

**결론**: 본인 사업자등록증으로 보유한 두 번호 (070 + 010) 라면 합법.

## 4.3 KCC 제출 서류

§2.1 의 필수 서류 + §2.2 조건부 서류 (해당 시) 와 동일.

추가 사항:
- **서비스 이용 목적서** (간단히 작성 — 양식 다운로드 가능)
- **준법 동의서** (msafer 페이지에서 체크박스로 동의)

## 4.4 온라인 신청 절차

**URL**: https://www.msafer.or.kr

### 단계별 진행:

1. **회원가입** → 사업자 계정 선택
2. **사업자 인증**: 사업자등록증 업로드 + 사업자등록번호 입력
3. 메뉴: **"발신번호 등록"** → **"신규 등록"**
4. **등록 번호 입력**:
   - 070-XXXX-XXXX (실제 SIP trunk)
   - 010-XXXX-XXXX (표시번호로 사용)
5. **이용 용도 작성**:
   ```
   AI 자동응답 시스템 운영
   - 부동산 임대 상담 비대면 응대
   - 비영업시간 (저녁/주말) 고객 문의 자동 처리
   - 임대료 알림 등 기존 고객 안내
   ```

7. **준법 동의 체크** + 제출
8. **접수번호 기록** (이후 KT 활성화 시 필요)

# 5. 준법 사항 (반드시 지켜야 할 부분)

발신번호 사전등록 시 다음을 약속하게 됩니다. **위반 시 등록 취소 + 과태료** 가능.

| # | 약속 사항 |
|---|---|
| 1 | 본인 명의 번호만 표시번호로 사용 |
| 2 | 등록한 용도 (AI 자동응답) 외 사용 금지 |
| 3 | 번호 변경 시 등록 정보 갱신 |
| 4 | 미신청 텔레마케팅 금지 (정보통신망법 별도 규정) |
| 5 | AI 통화 첫 문장에 자동응답 + 녹음 고지 |

**우리 시스템은 이미 #5 를 코드에 반영했습니다** (vipConfig.voice.recordingDisclosure — KR PIPA 준수). 나머지는 운영상 준수만 하면 됨.

## 5.1 통화 녹음 관련 (통신비밀보호법 §3)

- 통화 녹음은 **상대방에게 사전 고지** 필수
- 우리 AI 의 첫 안내:
  > "안녕하세요, 트리플H 부동산 AI 비서입니다.
  > 본 통화는 녹음되며 담당자에게 전달됩니다."
- 이 멘트 자체가 법적 고지 효과 발생

## 5.2 개인정보보호법 §15, §17

- 고객의 음성 + 전화번호 + 통화 내용 = 개인정보
- 수집·이용 목적: "AI 응답 + 상담 처리"
- 보관 기간: 30일 (우리 시스템 기본값)
- 30일 후 자동 삭제 (cron job 으로 처리)
- 고객 요청 시 즉시 삭제 (구두/이메일 요청 모두 인정)
